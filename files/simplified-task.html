Day 9: All in a Single Night

Santa needs to visit multiple locations and wants to find the shortest distance to travel between them.

He has a list of distances between each pair of locations.

Santa can start and end at any two different locations, but he must visit each location exactly once.

The task is to determine the shortest distance Santa can travel to achieve this.

For example, given the following distances:

London to Dublin = 464
London to Belfast = 518
Dublin to Belfast = 141

The possible routes are:

Dublin -> London -> Belfast = 982
London -> Dublin -> Belfast = 605
London -> Belfast -> Dublin = 659
Dublin -> Belfast -> London = 659
Belfast -> Dublin -> London = 605
Belfast -> London -> Dublin = 982

The shortest of these routes is London -> Dublin -> Belfast = 605, so the answer is 605 in this example.

Find the distance of the shortest route.
