Day 2: Inventory Management System

You have a list of box IDs in a warehouse. You need to find the checksum, which is calculated by counting the number of box IDs that contain exactly two of any letter, and the number of box IDs that contain exactly three of any letter. Multiply these two counts together to get the checksum.

For example, given the following box IDs:

  • abcdef
  • bababc
  • abbcde
  • abcccd
  • aabcdd
  • abcdee
  • ababab

There are four box IDs that contain a letter which appears exactly twice, and three box IDs that contain a letter which appears exactly three times. The checksum is calculated as 4 * 3 = 12.

What is the checksum for your list of box IDs?
